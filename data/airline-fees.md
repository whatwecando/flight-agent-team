# Base de connaissances — Frais cachés par compagnie

> Ce fichier est consulté par l'orchestrateur (CLAUDE.md) pour calculer le **Coût Total Réel** de chaque option de vol. Les montants sont des estimations basées sur les tarifs 2025-2026 et peuvent varier.

---

## Ultra Low-Cost

**Compagnies** : Ryanair, Wizz Air, Spirit, Frontier, Play, Volotea

| Frais | Montant estimé | Notes |
|-------|---------------|-------|
| Bagage cabine > 40×20×25cm | 20-45€ | Seul un petit sac sous le siège est "gratuit" |
| Bagage cabine 10kg ("Priority") | 15-30€ | Souvent bundlé avec embarquement prioritaire |
| Bagage soute 20kg | 25-50€ en ligne | +10-15€ de plus à l'aéroport |
| Choix de siège | 5-40€ | Attribution aléatoire sinon. Places avant et hublot plus chères |
| Embarquement prioritaire | 6-20€ | Souvent bundlé avec bagage cabine |
| Frais de paiement CB | 0-2% | Variable selon mode de paiement |
| Check-in à l'aéroport (Ryanair) | 55€ | Piège classique — toujours faire le check-in en ligne |
| **Surcoût moyen estimé** | **45-90€** | **Pour un passager avec 1 bagage soute** |

**Pièges courants :**
- Le prix affiché est quasi-nu — il ne comprend que le siège et un micro sac
- Ryanair facture le check-in à l'aéroport (55€) — toujours le faire en ligne
- Les tarifs bagages augmentent si ajoutés après la réservation
- Spirit/Frontier aux USA : même logique, ajoutez 40-80$ pour un bagage standard

## Low-Cost

**Compagnies** : EasyJet, Vueling, Transavia, Norwegian, Eurowings, Pegasus, IndiGo, AirAsia (court-courrier)

| Frais | Montant estimé | Notes |
|-------|---------------|-------|
| Bagage cabine | Inclus | Petit + grand bagage cabine généralement inclus |
| Bagage soute 23kg | 15-40€ en ligne | Moins cher réservé à l'avance. +15-25€ à l'aéroport |
| Choix de siège | 4-30€ | Places avant et issues de secours plus chères |
| Repas à bord | 5-15€ | Non inclus, achat à bord |
| Frais de paiement CB | 0-1.5% | |
| **Surcoût moyen estimé** | **20-45€** | **Pour un passager avec 1 bagage soute** |

**Pièges courants :**
- Le tarif de base inclut le bagage cabine mais PAS la soute
- Vueling/Transavia : vérifier si le "tarif Basic" inclut le grand bagage cabine
- Norwegian : les tarifs "LowFare" sont très restrictifs (pas de modif, pas de remboursement)

## Full-Service — Européens / Américains

**Compagnies** : Air France, Lufthansa, British Airways, KLM, Swiss, Iberia, SAS, Finnair, Delta, United, American Airlines

| Frais | Montant estimé | Notes |
|-------|---------------|-------|
| Bagage cabine | Inclus | Toujours inclus |
| Bagage soute 23kg — long-courrier | Inclus | En général 1 bagage 23kg inclus |
| Bagage soute 23kg — court-courrier | 0-60€ | **Gratuit en tarif standard, PAYANT en tarif "Light"** |
| Siège standard | Inclus | Siège avec plus d'espace : 10-25€ |
| Repas long-courrier | Inclus | |
| Repas court-courrier | Variable | Inclus AF/LH, payant BA/SAS sur certaines routes |
| Frais CB | 0% | |
| **Surcoût moyen** | **0-30€** | **Attention au tarif "Light"** |

**Piège principal — Tarif "Light" / "Basic" :**
- Air France "Light" : pas de bagage soute, pas de modification, pas de remboursement
- Lufthansa "Economy Light" : idem
- British Airways "Basic" : pas de bagage soute, siège attribué aléatoirement
- **Toujours vérifier si le tarif trouvé est "Light" ou "Standard"**

## Full-Service — Asie / Moyen-Orient

**Compagnies** : Emirates, Qatar Airways, Turkish Airlines, Singapore Airlines, ANA, JAL, Cathay Pacific, Korean Air, Thai Airways, EVA Air, China Airlines

| Frais | Montant estimé | Notes |
|-------|---------------|-------|
| Bagage cabine | Inclus | |
| Bagage soute | Inclus | 23-30kg selon compagnie (ANA/JAL : 2×23kg !) |
| Siège | Inclus | Choix gratuit ou peu cher |
| Repas | Inclus | Tous les repas inclus, même en éco |
| Entertainment | Inclus | Écrans individuels avec large catalogue |
| Frais CB | 0% | |
| **Surcoût moyen** | **0€** | **Tout est généralement inclus** |

**Avantages souvent ignorés :**
- Turkish Airlines via IST : souvent 15-25% moins cher que les vols directs européens, et transit fluide
- Qatar Airways via DOH : même logique, excellent hub
- Emirates via DXB : plus cher que Turkish/Qatar mais service supérieur
- ANA/JAL : 2 bagages soute de 23kg inclus en éco (rare !)
- Korean Air : 23kg soute inclus, excellent confort

## Frais OTA (agences de voyage en ligne)

| OTA | Frais de service | Pièges à surveiller |
|-----|-----------------|---------------------|
| **Site compagnie (direct)** | 0€ | **Toujours le comparer — souvent le même prix sans frais** |
| **Google Flights** | 0€ (redirige) | Agrégateur, pas de vente directe. Prix indicatifs |
| **Booking.com Flights** | 0-3€ | Prix compétitifs, support correct |
| **Trip.com** | 0-5€ | Bon pour l'Asie. Support variable |
| **Skyscanner** | 0€ (redirige) | Agrégateur, attention aux OTA obscures dans les résultats |
| **Kiwi.com** | 0-3% service | ⚠️ "Garantie correspondance" payante — ce ne sont PAS de vraies correspondances aériennes |
| **eDreams / Opodo** | 5-15€ | ⚠️ Abonnement "Prime" pré-coché à décocher |
| **Lastminute.com** | 5-20€ | Assurance pré-cochée à décocher |
| **GoToGate / Mytrip** | 3-10€ | Service après-vente médiocre |

**Règle d'or :** toujours vérifier le prix sur le site direct de la compagnie. L'écart avec l'OTA est souvent < 5€, et le SAV direct est infiniment meilleur.
